## [v0.1] - 2026-04-17

### Added
- `code-generator.md` - Generates production-ready code from Software Design Document (SDD)
- Domain structure with reusable coding agents

### Design Decisions

#### v0.1
- **Decision:** Create modular coding agents independent of any team
- **Reason:** Agents should be reusable across different team configurations

- **Decision:** Feature-based SOLID architecture as default
- **Reason:** Ensures maintainability and separation of concerns

- **Decision:** Clean Code standards are mandatory
- **Reason:** Generated code should be production-quality
